== Description ==

Simply insert your bing meta tag verification code using this helpful plugin.

Once your blog is verified, you can get your search statistics, Malware warning, Index status, HTML suggestion and many more bing webmaster features.

If you are using caching plugins like WP-Cache or WP Super Cache, you may need to temporarily clear or disable your cache for this plugin to work.

== Installation ==

1. Signup for Bing webmaster
2. Upload bing site verififaction plugin to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enter your verification id in plugin settings section as mentioned in screenshot.
5. Clear your cache if you are using anay cache plugin.
6. All Done.
